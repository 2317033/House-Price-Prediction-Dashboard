# 🏠 House Price Prediction Dashboard

## 📌 Project Overview

The **House Price Prediction Dashboard** is an interactive Machine Learning web application developed using **Python** and **Streamlit**. The dashboard enables users to explore housing data, visualize important trends, and predict house prices based on property features through an easy-to-use interface.

This project was developed as part of the **Horizon Intern – Machine Learning Internship Program**.

---

## 🎯 Objective

The primary objective of this project is to develop a Machine Learning model that can accurately predict house prices based on various property attributes such as area, number of bedrooms, bathrooms, parking facilities, furnishing status, and other amenities.

The project also demonstrates:

* Data Analysis
* Data Visualization
* Machine Learning Model Development
* Interactive Dashboard Creation
* Model Deployment using Streamlit

## ▶️ Installation & Execution

### Step 1: Clone Repository

```bash
https://github.com/2317033/House-Price-Prediction-Dashboard
```

### Step 2: Navigate to Project Folder

```bash
cd House-Price-Prediction
```

### Step 3: Install Required Libraries

```bash
pip install -r requirements.txt
```

### Step 4: Run Streamlit Application

```bash
streamlit run app.py
```
